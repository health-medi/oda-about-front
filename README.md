# ABOUT Front-end WEB

> 오다 소개 프론트엔드 웹

## Setup

Make sure to install the dependencies

```bash
yarn install
```

## Development

Start the development server on http://localhost:18031

```bash
yarn dev
```

## Production

Build the application for production:

```bash
yarn build
```
